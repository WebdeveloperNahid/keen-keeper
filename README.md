# Keen Keeper

##  Project Description
Keen Keeper is a simple and modern web application designed to help users manage and organize their tasks efficiently. It provides a clean interface and smooth user experience for better productivity.

🔗 Live Demo: https://keen-keeper-omega-ten.vercel.app/

---

## 🛠️ Technologies Used
- Next.js
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Vercel (Deployment)

---

## ✨ Key Features

-  **Task Management System** – Easily add, view, and manage tasks in an organized way.
-  **Fast & Responsive UI** – Fully responsive design optimized for all devices.
-  **Clean User Experience** – Simple interface focused on usability and performance.
